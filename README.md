# InterestCalculator
Calculating simple and Compound interest for a principal and term using express js and node

http://localhost:8090/home/percentage
Input:
{"amount":100,
	"interest":2,
	"term":2
}

Response:
{
    "maturitySimple": 104,
    "interest": 4,
    "compoundInterest": 4.039999999999992,
    "maturityCompound": 104.03999999999999
}
